# Ant Run

蚂蚁爬杆离线小游戏
杆子上有5只蚂蚁，当蚂蚁爬到杆子的端点就会掉落。由于杆子太细，两只蚂蚁相撞是，它们不能交错通过，只能各自反向爬回去。对每只蚂蚁我们只知道它距离杆子左边的距离，但不知道他的朝向，请计算所有蚂蚁落下杆子的最短时间和最长时间
附加：给出自定义每只蚂蚁的朝向，给出全部蚂蚁落下的时间。
